# GCS Gorilla Tag Fan Game Template

A Gorilla Tag Fan Game Template by Glitched Cat Studios!
This project was made in Unity 6.2 (6000.2.12f1), feel free to upgrade the version! (Don't downgrade it, it's a bad idea)

Note: When [MirrorVR](https://github.com/CodedImmersions/MirrorVR) releases, PhotonVR will be switched out for MirrorVR in this template.

## Installed
- [Unity 6000.2.12f1](https://unity.com/releases/editor/whats-new/6000.2.12f1)
- [Universal Render Pipeline 17.2](https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@17.2)
- [Meta XR Core SDK v81](https://developers.meta.com/horizon/downloads/package/meta-xr-core-sdk/81.0)
- [Meta XR Platform SDK v81](https://developers.meta.com/horizon/downloads/package/meta-xr-platform-sdk/81.0)
- [Photon PUN 2](https://assetstore.unity.com/packages/package/119922)
- [Photon Voice 2](https://assetstore.unity.com/packages/package/130518)
- [PhotonVR 0.0.4](https://github.com/fchb1239/PhotonVR)
- [PlayFab Unity SDK*](https://github.com/PlayFab/UnitySDK)
- [ParrelSync 1.5.3](https://github.com/VeriorPies/ParrelSync)

*The PlayFab EdEx package was removed due to security issues when having it in your project. To set your PlayFab Title ID, please go to `Assets/PlayFabSDK/Shared/Public/Resources` and open `PlayFabSharedSettings` in the inspector.

Enjoy!
